## Key Messaging Points

### What Makes Us Different

1. **AI Race Strategy** - "Personalized tactical analysis for every sailor - no one else offers this"
2. **Sailwave Integration** - "Works with your existing workflow - import/export .BLW files"
3. **Mobile-First** - "Native apps, not mobile websites - real push notifications, offline support"
4. **European Focus** - "Built for European sailing - multi-language, GDPR compliant, EU data"

### Handling Objections

**"We already use Sailwave"**
> "Great - so do thousands of clubs. RegattaFlow works WITH Sailwave. Import your .BLW files, add our mobile experience and AI features, export back to Sailwave anytime."

**"We don't have budget"**
> "The pilot is completely free. We're investing in European partnerships and want to prove value before asking for any commitment."

**"Our clubs won't change"**
> "They don't have to. Race committees can use Sailwave exactly as before. We add the competitor experience layer on top - mobile app, live results, AI coaching."

**"We tried something similar before"**
> "What didn't work? We're actively building based on class feedback - if there's a gap, tell us and we'll prioritize it."

**"We need to discuss with the board"**
> "Of course. Would it help if I prepared a one-page summary for the board? Or I'm happy to present directly if that's useful."
