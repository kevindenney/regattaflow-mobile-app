# BetterAt Multi-Org Goals (JHSON First)

## North-Star
- Deliver an end-to-end org learning and coaching demo across two orgs:
- JHSON (nursing)
- RHKYC (sail-racing)

## Success Criteria
- Org identity is stable and deduped in Learn.
- Join mode behavior is correct:
- JHSON open-join with domain gating.
- RHKYC request-to-join with approval flow.
- Realtime works for both personas:
- requester membership status updates live.
- requester receives live notification inserts.
- admin pending requests update live.
- Cohort and template assignment drives recommendation ordering.
- Interest context is reflected in labels and admin tooling.

## Primary Journeys
1. Requester searches org and requests access.
2. Admin approves/rejects request.
3. Requester sees realtime membership + notifications.
4. Admin manages cohorts, members, and templates.
5. Requester sees cohort-first recommendations.
