Foreword
When asked to write this book about racing tactics I was
concerned about creating another dry, repetitious volume
describing the standard tactics we constantly use on the
race course. I wanted to avoid an encyclopedic treatise
covering every possible type of tactics for every imagin-
able sailing situation. That has been attempted before and
makes very heavy reading. Moreover, its value is question-
able. Every tactical situation is slightly different; there-
fore it’s far better for the sailor to have a good foundation
of basic tactics and then improvise according to the cir-
cumstances that prevail at a particular time.
I avoided reading any other books on tactics so that I
wouldn’t be tempted to add a large number of tactics I
myself hadn’t experienced. Yet, because my style of writ-
ing is fairly tight (I have difficulty using a hundred words
to describe that which can be described in ten), I decided
